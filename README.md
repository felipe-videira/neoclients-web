# Neoclients Web
An client management app. Also check the [Neoclients API](https://github.com/felipe-videira/neoclients-api).

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
